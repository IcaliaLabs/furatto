Checkboxes and radios

Furatto helps you build beautiful checboxes and radio buttons by just adding the data attribute furatto and set it to the input type to style checkbox/radio. It uses the iCheck plugin by @damirfoy.

The color scheme for the checkboxes are: 

         
            <input type="checkbox" id="checkbox2" checked data-furatto='checkbox' data-color="dark">
            <label for="checkbox2" class="checkbox inline">Checkbox 2</label>

            <input type="radio" name="iCheck" id="radiobutton1" data-furatto='radio' data-color="orange">
            <label for="radiobutton1" class="checkbox inline">Radiobutton 1</label>
