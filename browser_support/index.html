Browser support

Furatto is developed to work best with the latest desktop and mobile browsers, meaning that legacy browsers might not render as nice as you might think (IE 7 and down), also some jQuery plugins may not work as expected.

Symbols meaning
  • Fully supported
  • Not supported
  • Not fully supported
Firefox Chrome IE Safari Opera
Android N/A N/A
iOS N/A
Mac OS X N/A
Windows

IE caveats

Some CSS3 properties may not display correctly(border-radius), as well as hardware acceleration CSS transitions.

A thought on caveats

We are working to give the best experience on every browser while keeping Furatto really lean. This means we will probably not add IE specific rules.
