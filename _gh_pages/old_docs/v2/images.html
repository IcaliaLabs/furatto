Images

A collection of images behaviors

Default image

To make the images in your site become responsive, just add the img class name.

Circular Image

If you ever want your images to be circles, just add the img-circular class name.

Polaroid Image

Old times never pass out, to get the polaroid effect on any image add the img-polaroid class.

Heads up!
The circular polaroid version for images need to be wrapped into a div element with the proper class names as shown in the example as a fix for iOS devices.

Selected Image

Whenever you want to highlight an image as if it was selected, you just need to add the img-selected class.

Heads up!
The circular selected version for images need to be wrapped into a div element with the proper class names as shown in the example as a fix for iOS devices.
